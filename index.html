CSS Regions Selection Use Cases

Use Case 1

  • Description: 1 named flow (with 1 source) in 1 region wrapped by 2 texts

  • Test

  • HTML:

    source-1 named-flow-1
text-1
region-1 named-flow-1
text-2

  • Visual:

    text-1
region-1 named-flow-1 source-1
text-2

Use Case 2

  • Description: 1 named flow (with 2 sources) in 1 region with some text in the middle of the 2 sources

  • Test

  • HTML:

    source-1 named-flow-1
text-1
source-2 named-flow-1
region-1 named-flow-1

  • Visual:

    text-1
region-1 named-flow-1 source-1 source-2

Use Case 3

  • Description: 1 named flow (with 1 source) splitted in 2 regions with some text in the middle of the regions

  • Test

  • HTML:

    source-1 named-flow-1
region-1 named-flow-1
text-1
region-2 named-flow-1

  • Visual:

    region-1 named-flow-1 source-1
text-1
region-2 named-flow-1 source-2

Use Case 4

  • Description: 2 named flows (each named flow with 1 source) in 2 regions with some text in the middle of the regions

  • Test

  • HTML:

    ource-1 named-flow-1
region-1 named-flow-1
text-1
region-2 named-flow-2
source-2 named-flow-2

  • Visual:

    region-1 named-flow-1 source-1
text-1
region-2 named-flow-2 source-2

Use Case 5

  • Description: 2 named flows (with 2 sources one of them nested) in 2 regions with a text in the middle

  • Test

  • HTML:

    source-1 named-flow-1
    source-2 named-flow-2 nested
region-1 named-flow-1
text-1
region-2 named-flow-2 nested

  • Visual:

    region-1 named-flow-1 source-1
text-1
region-2 named-flow-2 source-2 nested
