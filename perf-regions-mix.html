Performance test for selection in CSS Regions

Every time you click the Select button, the test performs a selection using JavaScript method setBaseAndExtent from word0 to . Then it shows the time requiered to execute the selection and calculates the average for all the selections done.

By default this test has 1000 divs (500 regular divs and 500 regions) but you can modify this number with a GET param. For example, for 2000 divs you should add ?2000 to the URL (see link).
