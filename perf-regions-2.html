Performance test for selection in CSS Regions with 2 flow threads

Every time you click the Select button, the test performs a selection using JavaScript methods collapse and extend from word0 to , passing through the rest of keywords. Then it shows the time required to execute the selection and calculates the average for all the selections done.

By default this test has 100 regions but you can modify this number with a GET param. For example, for 1000 regions you should add ?1000 to the URL (see link).
